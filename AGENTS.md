# AI Factory Agent Instructions

At the beginning of every new task in this repository, read `README.md` before
planning or editing. Then follow its documentation order:

1. `docs/architecture.md`
2. `docs/production_dataset.md`
3. `docs/validation_dataset.md`
4. `docs/certification.md`
5. `docs/code_guide.md`
6. `docs/cuda.md`

Treat those files and the automated tests as the current project contract.
`docs/journal.md` is historical and `docs/roadmap.md` is prospective; neither
overrides the current contract.

Before implementing a new model, curve, product, or result database, agree with
the user on the financial definition, parameter construction, numerical method,
and requested outputs. Then follow the complete production, validation,
notebook, and certification pipeline documented above.
